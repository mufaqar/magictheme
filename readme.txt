This is a basic WordPress theme starter using Bootstrap 5.
Files included:
- style.css
- functions.php
- header.php
- footer.php
- index.php
- page.php
- sidebar.php
- template-parts/content.php
- assets/ (empty folders for css, js, img)
Installation:
1. Upload the 'my-bootstrap-theme' folder into /wp-content/themes/
2. Activate from WordPress admin > Appearance > Themes
3. Create a menu and assign to 'Main Menu'
